# Contribuer au Collectif

Merci de vouloir aider ! Voici comment vous pouvez participer :

1. **Le Code :** Si vous êtes dev, aidez-nous à construire des outils décentralisés.
2. **Les Idées :** Ouvrez une "Issue" non pas pour un bug, mais pour proposer un sujet de débat ou une théorie alternative.
3. **Le Respect :** Nous sommes pacifistes. Les débats peuvent être vifs, mais jamais haineux. Nous acceptons les théories qui s'excluent mutuellement, car c'est du frottement des idées que naît la lumière.

*Soyez la cyanobactérie de votre environnement.*
